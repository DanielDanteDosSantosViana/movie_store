# movie_store
Loja de Filmes Online - Node.js+Angular+Mongodb

## Instruções ##

vá até a raiz do projeto e instale as dependências:
```
cd movie_store/
npm install

```
### Executando ###

na raiz do projeto vamos utilizar o nodemon para rodar

```
npm run nodemon
```
ou 
```
node app.js
```

### Acessando ###

http://localhost:8080
